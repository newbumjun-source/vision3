# Vision3 – AI 기반 유튜브 썸네일·제목 자동 생성 SaaS PRD

## 1. 제품 개요
대화형 챗 인터페이스 하나로 고CTR 유튜브 썸네일과 제목을 3분 내 자동 제작·후처리해 주는 멀티모달 SaaS.

## 2. 문제 정의
• 개인 크리에이터 다수는 Canva 등 수동 디자인 툴을 사용해 썸네일을 만들며 디자인 역량·시간 부족으로 CTR 손실 발생.
• 프롬프트 기반 AI 서비스는 복잡하고 결과 품질이 불안정함.

## 3. 핵심 가치 제안
1) 자연어 대화만으로 전문가 수준 썸네일·제목 완성  
2) 클릭률 최적화 로직이 숨겨져 있어 사용자는 ‘디자인 결정’ 스트레스 無  
3) 3분 내 결과물 제공으로 업로드 일정 가속화

## 4. 목표 사용자
• 개인 유튜브 크리에이터  
• 우선 타겟: ① 틈새 교육·지식 채널 ② 데일리 게이밍 채널 ③ 라이프스타일·엔터테인먼트 채널

## 5. 주요 사용 시나리오
새 영상 업로드 전, 챗으로 썸네일·제목을 빠르게 완성하고 즉시 게시.

## 6. 사용자 최종 목표
썸네일·제목 덕분에 영상 CTR을 높여 조회수를 늘린다.

## 7. 핵심 기능 (MVP)
1. 대화형 썸네일 생성 – Gemini Pro 기반 챗 UI
2. 자동 제목 생성 – 동일 대화 맥락에서 5개 후보 제안  
3. 다운로드/복사 & 사이즈 최적화 – 1280×720, 1920×1080, 1080x1920 선택  

## 8. 제외 범위 (MVP)
• 팀 협업, 풀 분석 대시보드, 템플릿 마켓플레이스, 다중 플랫폼(TikTok) 지원

## 9. 성공 지표 (KPIs)
• 평균 CTR 상승률 ≥ 15 %  
• 썸네일 생성→채택 비율 ≥ 50 %  
• 30일 재사용률 ≥ 40 %  
• 최초 유료 전환율 ≥ 8 %

## 10. 기능 요구 사항
FR-01 사용자는 채팅으로 영상 주제·톤·희망 스타일을 입력할 수 있다.     
FR-01 모든 프롬프트·내부 디자인 로직은 사용자에게 노출되지 않는다.

## 11. 비기능 요구 사항  
• 플랫폼: 반응형 웹(모바일 최적화)  
• 보안: Supabase ‑ Row Level Security, OAuth(Clerk)  
• 가용성: 99.5 % 이상  
• 개인정보: 채팅 내용 30일 후 자동 익명화

## 12. 기술 스택
Next.js 15, TypeScript, Supabase, Clerk, @tanstack/react-query, shadcn UI, lucide-react, LemonSqueezy(결제)
